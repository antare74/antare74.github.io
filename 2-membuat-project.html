Membuat Project Laravel

Anda dapat meletakkan Folder Project di manapun, disini saya akan memasukkan Project saya di Document/namaProject jadi saya jalankan perintah cd /Document/namaProject

Karena posisi kita sedang pada direktori Document/namaProject maka langsung kita eksekusi saja projectnya.

Untuk mengantisipasi terjadinya corrupt file, pakai perintah dibawah ini untuk membuat project:

composer create-project --prefer-dist laravel/laravel namaProject

Server Lokal

Untuk menjalankan laravel anda dapat menggunakan local server degan mengetikan php-artisan serve pada direktori anda membuat "Project" laravel tadi, contoh:

buka cmd lalu:

cd /home/Documents/namaProject

output

Laravel development server started: 127.0.0.1:8000

lalu copy 127.0.0.1:8000 dan buka pada browser anda.

NOTE!

Jika ada file corrupt atau kendala saat installasi laravel, ikuti langkah dibawah:

  1. Install package php zip

    sudo apt-get install php7.2-zip

    lalu coba buat proect baru kembali, tetapi apabila masih terdapat error maka anda perlu menginstall packagenya satu demi satu.

  2. Ikuti langkah dibawah ini:

    sudo apt-get install BCMath7.2-zip

    sudo apt-get install Extension7.2-zip

    sudo apt-get install JSON7.2-zip

    sudo apt-get install Mbstring7.2-zip

    sudo apt-get install OpenSSL7.2-zip

    sudo apt-get install PDO7.2-zip

    sudo apt-get install Tokenizer7.2-zip

    sudo apt-get install XML7.2-zip

    Lalu buat project anda kembali, semoga tidak ada yang error :)
